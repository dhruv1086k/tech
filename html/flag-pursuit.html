Description :
Join us at Tech Samaaroh 4.0 for 'Flag Pursuit', an exhilarating Capture the Flag event! Test your skills in cybersecurity and programming as you navigate through challenges. Explore two login panels, uncovering usernames and passwords through website infiltration and SQL injection. Delve deeper into Node.js API keys, Python code analysis, and navigating Metasploit machines over the network. Sharpen your skills in cryptography and unravel hidden messages in steganography challenges.
Rounds :
Participants progress through 7 rounds, including website login, SQL Injection, API key decryption, Python code analysis, Metasploit network penetration, cryptography, and steganography. The entire competition spans three hours encompassing all rounds in a single continuous session.
